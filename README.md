kolib is the core library that will power Koli apps.

Koli is an archive reader app that lets users store their data exported from various platforms. e.g. requested archive from X/Twitter.

Instead of having bunch of json, html, and similar files, kolib parses and saves them in a sqlite database in a folder users choose, and provides an API for developers to build applications with, so that users can view their messages, posts, bookmarks, etc. in a familiar UI, rather than from different formats of files that are hard to read through.
